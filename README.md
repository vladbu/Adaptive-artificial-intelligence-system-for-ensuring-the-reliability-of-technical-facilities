

# Адаптивная система ИИ для обеспечения надёжности технических объектов

![Python](https://img.shields.io/badge/python-3.10+-blue.svg)
![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?logo=pytorch&logoColor=white)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Интеллектуальная гибридная система предиктивного мониторинга и диагностики состояний **силовых масляных трансформаторов**. Проект реализует технологический конвейер (pipeline), объединяющий глубокое обучение для прогнозирования временных рядов и вероятностное моделирование для классификации аварийных состояний.

## 📋 Описание проекта

В отличие от традиционных систем SCADA, работающих по жестким пороговым значениям и сигнализирующих о проблеме только «после факта» её наступления, данная система нацелена на выявление скрытых нелинейных зависимостей и **заблаговременное предупреждение** об аварии.

### Ключевые особенности:
*   **Гибридная архитектура:** Связка **LSTM** (прогноз состояния на шаг $n+1$) и **GMM** (вероятностная классификация прогноза).
*   **Высокая точность:** Прогнозирование — **90%**, индикация аварийных зон — **97%**.
*   **Адаптивность:** Поддержка **дообучения (fine-tuning)** под индивидуальные тепловые профили конкретных моделей трансформаторов при наличии исторических данных телеметрии (нагрузка HUFL/MUFL/LUFL и температура масла OT).
*   **Диагностика 9 типов неисправностей:** Включая межвитковые замыкания, увлажнение изоляции, старение масла и повреждения системы охлаждения.

## 🛠 Технологический стек

*   **Язык:** Python
*   **Библиотеки:** 
    *   `PyTorch` — разработка архитектуры LSTM (47 533 параметра).
    *   `Scikit-learn` — реализация Gaussian Mixture Model.
    *   `Optuna` (алгоритм TPE) — автоматическая оптимизация гиперпараметров.
    *   `Pandas`, `Numpy` — предобработка данных и анализ временных рядов.
    *   `Matplotlib`, `Seaborn` — визуализация результатов и матриц корреляции.

## 📊 Данные

Для обучения и тестирования использовался набор данных **Electricity Transformer Dataset (ETDataset)**, предоставленный Beijing Guowang Fuda Science & Technology Development Company. Датасет содержит двухлетние записи телеметрии силовых трансформаторов с часовым интервалом.

## 📂 Структура репозитория

*   `final.py` — основной скрипт работы программы (прогноз + индикация).
*   `eda.py` — исследовательский анализ данных и визуализация зависимостей.
*   `final_best_weights.pth` — обученные веса нейросети LSTM.
*   `trained_gmm_model.pkl` — сохраненная модель кластеризации GMM.
*   `scaler.pkl` — параметры нормализации данных (MinMaxScaler).
*   `best_gmm_params.json` — оптимальные гиперпараметры, подобранные Optuna.

## 🚀 Использование

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/vladbu/Adaptive-artificial-intelligence-system-for-ensuring-the-reliability-of-technical-facilities.git
    ```
2.  Установите зависимости:
    ```bash
    pip install torch scikit-learn pandas matplotlib optuna joblib
    ```
3.  Запустите основной скрипт для получения прогноза и оценки рисков:
    ```bash
    python final.py
    ```
